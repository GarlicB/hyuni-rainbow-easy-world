# 현이의 무지개 친구 월드

8세 아이가 갤럭시탭에서 쉽게 놀 수 있도록 만든 3D 웹게임입니다. Rainbow Friends의 수집, 상자 숨기, 컬러 친구 분위기를 참고했지만 캐릭터와 이름은 오리지널로 만들었습니다.

## 특징

- 게임오버 없음: 친구에게 닿아도 죽지 않고 골드 선물을 받습니다.
- 빠른 이지 액션: 이동과 점프를 빠르게 조정했고, 죽지 않는 대신 몬스터가 밀쳐냅니다.
- 확장된 밤 맵: 더 넓은 월드, 푸른 은하수 하늘, 먼 산, 추가 조명, 큰 나무와 트램폴린 안전지대가 있습니다.
- 레벨 진행: 블록, 음식, 퓨즈, 배터리 목표를 모으며 4단계를 통과합니다.
- 몬스터 전투: 파랑, 초록, 주황, 보라, 노랑, 청록이 각자 다른 방식으로 공격하고, 공격 버튼으로 처치할 수 있습니다.
- 충돌 반동: 몬스터와 붙으면 서로 밀려나고, 공격 성공 직후에는 짧게 묵직해져 타격감이 납니다.
- 블록 물리: 빠르게 달려도 블록을 관통하지 않고, 부딪히면 미끄러짐과 반동 모션이 납니다.
- 무기 성장/전환: 1탄은 밀치기, 2탄부터 활과 화살, 3탄부터 파이어볼을 사용하고 열린 무기 안에서 바꿀 수 있습니다.
- 활 충전: 활은 누르고 오래 당길수록 더 멀리, 더 세게 날아갑니다.
- 노랑 낚아채기: 날개 달린 노랑에게 잡히면 잠깐 공중에서 방향 조절을 하고, 트램폴린에 착지하면 충격을 피합니다.
- 나무 보상: 나무에 부딪히면 골드, 벌레몬스터, 희귀 별 보상이 나올 수 있습니다.
- 다음 문 포털: 레벨 조건을 끝내면 백현이 앞에 다음 레벨 문이 열립니다.
- 골드 상점: 실제 결제 없이 게임 안에서 모은 골드로 아이템과 색을 엽니다.
- 아바타 편집: 현이 캐릭터의 피부, 옷, 머리, 액세서리를 게임 안에서 바로 바꿀 수 있습니다.
- 갤럭시탭 대응: 화면 버튼, 조작 도움, 미니맵, 가로 모드, 홈 화면 설치용 PWA 설정이 포함되어 있습니다.

## 실행

```bash
npm install
npm run dev
```

## 설치형 웹앱으로 확인

```bash
npm run build
npm run preview
```

갤럭시탭 Chrome에서 preview 주소를 연 뒤 브라우저 메뉴의 홈 화면 추가를 사용하면 앱처럼 열 수 있습니다.

## 조작

- 이동: `WASD`, 방향키, 또는 화면 왼쪽 조이스틱
- 카메라: 화면 오른쪽 빈 곳 드래그
- 점프: `Space` 또는 `▲`
- 공격: `F` 또는 `⚡`, 활은 길게 눌렀다가 떼기
- 무기 변경: `R`, 숫자 `1`/`2`/`3`, 또는 `↻`
- 상자: `B` 또는 `▣`
- 블록 놓기: `E` 또는 블록 버튼
- 블록 지우기: `Q`/`Backspace` 또는 `×`
- 아바타: 화면 오른쪽 위 `아바타` 버튼
- 도움말: 화면 오른쪽 위 `도움` 버튼
